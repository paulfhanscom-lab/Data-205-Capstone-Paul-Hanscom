# Data-205-Capstone-Paul-Hanscom
## Overview
This Project explores the relationship between crash data and weather data in montgomery county in 2025. The goal being to analyse patterns and relationships in order to show the danger of driving in hazardous weather.
## Tools used
* R(RStudio)
* gstat (R library)
* kriging (spacial interpolation)
* leaflet (R library)
## Project Structure
This repository contains
* data/ - raw and processed data
* ingestion/ - code for data processing
* eda/ - exploratory data analysis
* analysis/ - modeling and visualizing 
* reports/ - final outcomes and presentation
